Funcionalidades

+ping
Este comand o faz o bot reagir a sua mensagem com 3 emojis, escolha um deles para obter o poder.
+aula
Este co mando manda o link da sua aula do itinerário.
+lvl
Este co mando retorna as informações do seu level.
+x1
Este comand o te permite tirar um X1 com seu amigo.
+chegae
Este comando faz eu entrar na call e tocar uma musica pog!
+info
Este co mando retorna as informações do usuário mencionado.
